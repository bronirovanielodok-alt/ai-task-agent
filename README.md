# AI Task Agent

Простой AI-агент на Python, который принимает задачи от пользователя и выполняет их через ИИ (OpenAI GPT).

## Возможности

- Принимает текстовые задачи на естественном языке
- Анализирует задачу и создаёт план выполнения
- Выполняет задачу и возвращает результат
- Работает в интерактивном режиме

## Установка

1. Клонируйте репозиторий или перейдите в папку проекта:
```bash
cd ai_task_agent
```

2. Создайте виртуальное окружение (рекомендуется):
```bash
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate     # Windows
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Настройка

1. Получите API ключ на [OpenAI Platform](https://platform.openai.com/)

2. Установите переменную окружения:
```bash
# Linux/Mac
export OPENAI_API_KEY="ваш-api-ключ"

# Windows (CMD)
set OPENAI_API_KEY=ваш-api-ключ

# Windows (PowerShell)
$env:OPENAI_API_KEY="ваш-api-ключ"
```

## Запуск

```bash
python -m src.agent
```

## Пример использования

```
==================================================
AI Task Agent - Выполнение задач через ИИ
==================================================

Для выхода введите 'quit' или 'exit'

Задача: Напиши короткое стихотворение про кота

Выполняю задачу...

Результат:
Котёнок мягкий, словно пух,
Легко прыгает вокруг,
Глазки блестят в темноте,
Любит он поспать в теплоте.
```
